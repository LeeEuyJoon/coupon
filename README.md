# Coupon

프로덕션 수준의 대량 포인트 쿠폰 생성·사용 시스템

---

## 개요

캠페인 단위로 최대 100만 개의 포인트 충전 쿠폰을 안정적으로 생성하고,
사용자가 쿠폰 코드를 입력하면 포인트가 즉시 충전되는 시스템

단순히 기능을 구현하는 데 그치지 않고, 대규모 데이터 처리·코드 보안·동시성 제어·성능 최적화·인프라 확장까지
프로덕션 환경에서 실제로 요구되는 문제들을 단계적으로 해결하는 것을 목표로 함

---


## 주요 기능

- **캠페인 생성**: 포인트 금액별 쿠폰 수량을 설정한 캠페인을 생성
- **쿠폰 생성**: 캠페인에 설정된 조건에 따라 고유한 쿠폰 코드를 발급
- **쿠폰 사용**: 유효한 쿠폰 코드를 입력하면 해당 포인트가 사용자 잔액에 충전
- **포인트 충전 이력**: 쿠폰 사용 내역 기록 및 조회
